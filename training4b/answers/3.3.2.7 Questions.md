# Part 3: Configure WPA2 RADIUS for Metropolis Bank HQ
4. When considering a large organization, why is WPA2 RADIUS more beneficial than WPA2 PSK?
Karena WPA2 RADIUS (Enterprise) memberikan autentikasi per user dengan username & password unik, dikelola secara terpusat melalui server AAA/RADIUS. Keuntungannya:
- Lebih aman dibanding satu shared key (PSK)
- Mudah mencabut akses user tertentu tanpa mengubah seluruh jaringan
- Lebih skalabel dan cocok untuk organisasi besar dengan banyak pengguna
