# Part 1: Connect to the Web Server
1. Why would a user be concerned when submitting information using this website?
Karena situs menerima data login lewat HTTP yang tidak terenkripsi, sehingga informasi yang dikirim (mis. username/password) bisa disadap oleh pihak ketiga.

2. Why would a user be less concerned when submitting information using this website?
Karena situs menggunakan HTTPS (SSL/TLS) sehingga data autentikasi dienkripsi saat dikirim, mengurangi risiko penyadapan.

# Part 2: Prevent Unencrypted HTTP Sessions
3. Is Sally’s computer able to access the HQ Internet Web Server using HTTP?
Tidak, HTTP diblokir oleh konfigurasi ACL pada router sehingga permintaan HTTP tidak berhasil terhubung.

4. Is Sally’s computer able to access the HQ Internet Web Server using HTTPS?
Ya, HTTPS tetap dapat diakses, permintaan HTTPS berhasil terhubung ke server.
